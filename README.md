## üèõÔ∏è Thesis
Modern digital environments are engineered for extraction, fragmenting user attention through high-entropy visual noise and dopamine loops. **The Sovereign OS** is a browser-integrated "Void" designed to counter this.

It replaces the default "New Tab" experience with a high-aesthetic, low-entropy terminal that forces **Single-Task Intent**. It utilizes **Adlerian Psychology** (teleology/purpose-driven action) and **Stoic constraints** to eliminate "shiny object syndrome" and enforce deep work.

## ‚ö° Functional Pillars

### 1. The Void (Visual Silence)
A pure black (`#000000`), distraction-free interface that resets cognitive load. No icons, no feeds, no links.

### 2. Intent Locking (The Guardrail)
Users must explicitly declare their immediate intent. Once declared, the interface **locks**. There is no "edit" or "back" button. The only way out is through completion.

### 3. Temporal Feedback (The Ticking Clock)
A dedicated session timer tracks the exact duration of the current "Deep Work" state, providing immediate quantitative feedback on agency regained.


2.  **Install Dependencies**
    ```bash

    ```

3.  **Ignite the Engine**
    ```bash
    npm run dev
    ```

4.  **Access the Terminal**
    Open `http://localhost:5173` in your browser.
